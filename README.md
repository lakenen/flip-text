# flip-text

A node module and CLI for flipping text upside-down.

## Install

```
npm install -g flip-text
```


## Run

In JS:
```js
var flip = require('flip-text')

flip('Beep boop, BOP?')
// ¿ԀO𐐒 'dooq dǝǝ𐐒
```

Or CLI:
```
flip "Beep boop, BOP?"
```
